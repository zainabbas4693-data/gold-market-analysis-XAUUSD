# Gold Market Analysis (XAU/USD)

## 📊 Project Overview
This project analyzes historical gold prices (XAU/USD), focusing on trends, volatility, and relationship with the US Dollar Index (DXY).

## 🎯 Objectives
- Identify long-term trends in gold prices
- Analyze volatility patterns
- Study relationship between gold and USD

## 🛠 Tools Used
- Excel (Data Cleaning & Analysis)
- Tableau (Visualization & Dashboard)

## 🔍 Key Insights
- Gold shows a strong long-term upward trend
- Volatility occurs in episodic spikes
- Relationship between gold and USD is dynamic, not strictly inverse
- Returns analysis shows moderate relationship (R² ≈ 0.4)

## 📈 Analysis Performed
- Daily returns calculation
- Moving averages (7-day)
- Volatility measurement
- Correlation analysis
